InfinityShopping is the e-commerce shop generator. Choose from 21 store templates (everything will change in the whole design, buttons, background, etc.) then design the home page and an e-commerce shop is going to be yours. In the future, InfinityShopping will be generating an e-commerce shop in one command and will be available in jhipster marketplace. At the moment is available the development version of InfinityShopping there is on develop branch. You can download InfinityShopping from our GitHub and add more addons from Jhipster marketplace or change something in yo-rc.json file as in an existing Jhipster project.
